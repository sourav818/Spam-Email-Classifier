# Spam Email Classifier

This project uses Machine Learning and Natural Language Processing (NLP) to classify messages as **Spam** or **Not Spam**.

## Project Overview
The model is trained on a labeled SMS dataset and uses the **Naive Bayes** algorithm with **TF-IDF** vectorization to detect spam messages.

## Features
- Detects spam messages  
- High accuracy  
- Real-time message testing  
- Simple and lightweight  

## Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- NLP (TF-IDF)  

## How to Run

1. Install dependencies  
pip install -r requirements.txt


2. Run the program  


python spam_classifier.py


3. Enter a message to test  
Type `exit` to stop.

## Example

Input:


Win ₹10,000 now! Click here!


Output:


SPAM ❌
